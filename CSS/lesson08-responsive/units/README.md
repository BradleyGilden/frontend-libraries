# CSS Units

units are essential for responsive design, these are just quick notes on CSS units

- **Absolute Units**
  - size never changes
    - Pixels (px)
    - pt, cm, mm, in, etc
- **Percentages**
  - Relative to their parent (except on height)
    - %
- **Relative Units**
  - Relative to font-size
    - em, rem
  - Relative to viewport
    - vw, vh, vmin, vmax
